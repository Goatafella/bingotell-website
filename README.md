# Bingotell Website

Upload all files and folders in this package to the root of a public GitHub repository. Enable GitHub Pages from **Settings → Pages → Deploy from a branch → main → /(root)**.

Use these URLs in Play Console:
- Website: `https://YOUR-USERNAME.github.io/bingotell-website/`
- Privacy policy: `https://YOUR-USERNAME.github.io/bingotell-website/privacy.html`
- Support: `https://YOUR-USERNAME.github.io/bingotell-website/support.html`
- Terms: `https://YOUR-USERNAME.github.io/bingotell-website/terms.html`
